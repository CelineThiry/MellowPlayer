MellowPlayer

Cloud music integration for your desktop!

Download .zip Download .tar.gz View on GitHub

MellowPlayer is a free, open source and cross-platform desktop application that runs web-based music streaming services in its own window and provides integration with your desktop (hotkeys, multimedia keys, system tray, notifications and more).

MellowPlayer is a Qt based alternative to NuvolaPlayer, specifically crafted for KaOS.

License

MellowPlayer is licensed under the GPL license.

Features

  • Grooveshark integration
  • Deezer integration
  • Cross-platform (works on Windows, Mac OSX and Linux)
  • System tray integration (minimize to tray on close, notifications)
  • Global shortcuts support (Media Keys or any custom shortcut)
  • Mpris2 Interface (Linux only)
  • Plugin based application (you can extend the app or add support for new service by writing a plugin)
